Primary Standard

ISO/IEC 18004

The QR Code model, placement rules, format/version information, and capacity relationships ultimately derive from the ISO/IEC 18004 specification family. QR Peach uses public summaries and implementation behavior as the accessible layer over that standard.

2
Vendor Reference

Denso Wave QR Code Essentials

Denso Wave's public documentation provides concise explanations of finder patterns, alignment patterns, timing patterns, error correction levels, and general QR usage conventions.

3
Capacity Tables

Thonky QR Code Tutorial

Thonky's QR tutorial is one of the clearest public references for version capacities, Reed-Solomon block layouts, format strings, masking, and placement examples. It is especially useful as a sanity check against implementation code.

4
Project Runtime

qrpeach.js

The runtime is implemented for QR Peach against the QR specification and embeds the generation tables and source metadata directly in the library.
